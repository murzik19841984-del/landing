Лендинг эксперта по управленческому учету

Профессиональный одностраничный сайт (лендинг) для эксперта по управленческому учету Марии Карповой.

Особенности

Стек: Чистый HTML5, CSS3 и Vanilla JavaScript.

Дизайн: Минималистичный стиль «финансового инжиниринга» с использованием шрифта JetBrains Mono.

Адаптивность: Полная поддержка мобильных устройств (iOS/Android), планшетов и десктопов.

Производительность: Один файл index.html (все стили и скрипты внутри), быстрая загрузка без внешних библиотек.

Интерактивы: Анимация появления блоков при скролле, карусель услуг, адаптивное видео.

Как запустить

Склонируйте репозиторий или просто скачайте файл index.html.

Откройте index.html в любом современном браузере.

Структура репозитория

index.html — основной и единственный файл сайта.

README.md — краткая документация проекта.

Разработано для Марии Карповой.
